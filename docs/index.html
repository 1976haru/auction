경매·공매 지능형 에이전트

이 앱은 무엇?

법원경매·온비드 공매 매물을 검색·필터·차트·자연어 로 빠르게 탐색하는 정적 데모입니다. PWA 로 홈 화면에 설치하면 standalone 앱처럼 동작해요.

어떻게 쓰나

  1. 검색바 에 주소·사건번호·키워드 입력 (예: 강남, 2025타경, 임차인)
  2. 빠른 메뉴 칩 또는 프리셋 칩 으로 한 번에 좁히기 (예: 🚀 고수익 저위험)
  3. 매물 카드 탭 → 상세 모달에서 시세 트렌드, 입찰가 시뮬레이터, 점수 분해, 위험·체크리스트 확인
  4. ★ 관심 · 📝 메모 로 매물별 정리, ⇆ 비교 트레이로 2~5개 나란히 보기
  5. 🔗 공유 로 매물·결과를 카톡 등으로 (받은 사람도 같은 화면)
  6. ⚙ 설정 에서 데이터 백업/복원, 🌙 다크 모드, 🖨 인쇄/PDF

데이터 · 보안

  • 모든 매물은 mock 데이터 — 실제 거래 대상이 아닙니다.
  • 실 법원경매·온비드·국토부 API 호출, 실제 입찰·금전거래·외부 전송은 하지 않습니다.
  • 관심·메모·비교·검색어·본 이력은 자기 폰 브라우저(localStorage) 에만 저장 — 서버로 전송 없음.
  • 외부 라이브러리 0, 외부 API 호출 0 (mock JSON 만 fetch).
  • 📡 RSS 피드: feed.xml — 추천 점수순 상위 50건을 외부 피드 리더로 구독 가능.

면책

권리분석 결과는 참고용 위험 체크리스트와 추가 확인사항으로만 제공되며, 법률·투자 판단을 단정하지 않습니다. 실제 입찰 전에는 등기부등본·전입세대열람·현장조사·전문가 자문이 필요합니다.

설정 · 데이터 백업

저장된 내 데이터

이 데이터는 자기 폰 브라우저(localStorage) 에만 저장돼요. 캐시 삭제 / 폰 변경 시 사라지므로 가끔 백업해 두는 게 안전합니다.

백업 (다운로드)

관심 · 메모 · 비교 · 최근 검색어 · 테마를 한 JSON 파일로 저장합니다.

복원 (업로드)

백업 JSON 파일을 불러옵니다. 기본은 덮어쓰기 이며, '추가' 모드는 기존 데이터에 합칩니다.

📝 내 메모 모아보기

메모를 적어둔 모든 매물을 한 곳에서 검색·확인합니다. 항목 클릭 시 그 매물의 상세 모달을 엽니다.

전체 지우기

관심·메모·비교·최근 검색어를 한 번에 비웁니다. 백업해 두지 않으면 복구 불가.

키보드 단축키

/통합 검색창 포커스
r검색·필터 초기화
t라이트/다크 모드 전환

← →상세 모달 안에서 이전/다음 매물
f상세 모달 안에서 ★ 관심 토글

입력창에 포커스가 있을 때는 단축키가 동작하지 않아요. 모바일에선 키보드가 없으니 화면의 검색바·필터·칩·트레이로 같은 동작이 가능합니다.
